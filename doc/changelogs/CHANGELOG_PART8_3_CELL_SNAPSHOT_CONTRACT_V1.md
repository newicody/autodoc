# Changelog — Part 8.3 Cell Snapshot Contract v1

## Added

- Immutable `CellSnapshot` value object.
- Schema constant `missipy.cell.v1`.
- JSON dict and JSON line serialization.
- Strict validation for schema, lifecycle state, finite score, non-negative age and cost.
- Contract documentation and tests.

## Not changed

- No Scheduler path.
- No EventBus path.
- No recorder path.
- No renderer path.
- No dependency addition.
