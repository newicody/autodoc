# Specialists/laboratories chain closure audit — 0284-r8

## Purpose

The audit separates **implémentation complète** from **preuve opérationnelle**.
A repository may contain every contract and composition while still lacking a
real correlated run on the current machine. Phase 0284 is not declared closed
as an integrated capability until the real evidence is green.

## Observation

The EventBus remains observation-only. The existing Cell Lens bridge may append
`missipy.cell.v1` records and VisPy may tail them, but this path cannot command
the Scheduler or laboratories.

## Project ownership

Autodoc has no project-management mode. Active workflow, Issue forms, view
configuration and mutation adapters belong to the bundle copied into
`newicody/projects`. Active project-management surfaces under Autodoc's root
`.github/` block closure.

## Closure states

```text
red        missing/incomplete/forbidden implementation surface
transition implementation complete, real correlated evidence not supplied
green      implementation complete and all real evidence verified
```

The audit patch itself is documentary/audit work and therefore reports
`live_path_status: n/a`. Its result object may report `green` only from supplied
real execution evidence.
