# Copilot advisory ProjectV2 publication closure

```text
GitHub Actions run
├── authoritative request
├── Copilot advisory
└── dual artifact manifest
        ↓ download locally
digest and correlation validation
        ↓
publication preview
        ↓ operator approve
ProjectV2 projection preview
        ↓ exact plan digest + two locks
ProjectV2 field mutation
        ↓
query-only relecture / Copilot view
```

The workflow is deliberately unchanged. It cannot call the projection adapter.
The local operator supplies the Issue number, policy decision and approval.

The authoritative request remains authoritative. Copilot is an untrusted hint.
