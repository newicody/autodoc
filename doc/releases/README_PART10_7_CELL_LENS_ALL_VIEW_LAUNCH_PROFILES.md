# Part 10.7 — Cell Lens All View Launch Profiles

Part 10.7 adds a helper that prints all local read-only launch profiles for the
Cell Lens demo.

## Added

- VisPy desktop launch profile.
- Browser Canvas launch profile.
- Browser health Canvas launch profile.
- SSE stream launch profile.
- Tests and docs.

## Not added

- No server implementation.
- No renderer implementation.
- No command channel.
- No dependency.
