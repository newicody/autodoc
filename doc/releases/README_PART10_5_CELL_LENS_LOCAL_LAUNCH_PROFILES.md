# Part 10.5 — Cell Lens Local Launch Profiles

Part 10.5 adds a small helper that prints the local launch profiles for the
Cell Lens demo.

## Added

- VisPy desktop launch profile.
- Browser Canvas launch profile.
- SSE stream launch profile.
- Wayland/XCB Qt platform detection helper.
- Tests and docs.

## Not added

- No server implementation.
- No renderer implementation.
- No command channel.
- No dependency.
