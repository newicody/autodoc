# Part 9.2 — Cell Observation Event Adapter

Part 9.2 prepares the handoff from synthetic data to real recorded observation
data without connecting to the live bus yet.

## Added

- `missipy.cell_observation_event.v1`
- Conversion to `missipy.cell.v1`
- Local JSONL conversion tool
- Tests and documentation

## Not added

- No live bus subscription
- No recorder core mutation
- No Scheduler path
- No VisPy change
- No mobile SSE
