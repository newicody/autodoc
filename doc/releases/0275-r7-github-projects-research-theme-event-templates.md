# 0275-r7 — GitHub Projects research/theme/event templates

This phase provides an installable bundle for the private management repository
`newicody/projects`.

The bundle supports durable research tickets, optional theme containers,
cross-theme independent events, source selection, media drop zones, next-cycle
context exclusions and optional external-inference intent.

The workflow is explicit and read-only. Automatic `Status` transition detection
remains deferred to 0275-r10.
