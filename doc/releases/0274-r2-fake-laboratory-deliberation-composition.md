# 0274-r2 — Fake laboratory deliberation composition

The deterministic fake laboratory now participates in the existing
server-oriented deliberation cycle.

Every requested specialist is visited through the one platform Scheduler. The
results become preliminary opinions, optional refined demands, a deliberation
round, normalized fragments and a local liaison synthesis.

A `FinalArtifactEnvelope` is created only when every visit completed and the
round converged. Remote publication still requires the existing operator gate.
