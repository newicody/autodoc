# 0275-r6 — GitHub research Kanban operator model

This documentary phase recenters the GitHub Project on the human operator.

The Board uses:

- optional horizontal `Thème` groups;
- durable research tickets as cards;
- `Étape` as the column and operator intention;
- local immutable cycles for repeated execution of the same ticket;
- explicit related research, media, external inference and context exclusions.

The phase also documents safe cleanup and preparatory installation. It does
not yet install column-trigger processing or OpenRC.
