# Part 10.1 — Cell Snapshot SSE Stream Contract

Part 10.1 defines the read-only SSE text contract for streaming
`missipy.cell.v1` snapshots to a future local browser view.

## Added

- SSE event contract.
- Journal-to-SSE dry-run helper.
- CLI for writing SSE text from a journal.
- Tests and docs.

## Not added

- No local server.
- No endpoint.
- No browser UI.
- No command channel.
- No optimization loop.
