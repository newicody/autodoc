# Part 8.6 — Synthetic Cell Population Generator

Part 8.6 adds a deterministic synthetic generator for `missipy.cell.v1`
snapshots.

## Added

- Synthetic source class profiles.
- Deterministic synthetic cell seed generation.
- Synthetic snapshot generator.
- CLI for generating a JSONL journal.
- Tests and contract documentation.

## Not added

- No EventBus consumer.
- No recorder stream integration.
- No VisPy.
- No mobile SSE.
- No optimization loop.
