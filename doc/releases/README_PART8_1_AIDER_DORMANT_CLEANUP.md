# Part 8.1 — Aider Dormant Cleanup

This patch adds a cleanup tool and documentation for putting Aider in dormant
mode.

## Scope

- Keep Roadmap B active.
- Keep patch queue workflow active.
- Clean generated Aider artifacts.
- Document that Aider/API is paused.

## Out of scope

- No autonomous agent.
- No API integration.
- No GitHub mutation.
- No Scheduler change.
- No DOT.
- No SVG.
